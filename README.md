# Number Guessing Game (Python)

A simple beginner-friendly Python game where the user guesses a randomly generated number.

## Features
- Random number between 1 and 100
- User input with validation
- Shows "Too high" or "Too low" hints
- Counts number of attempts
- Handles invalid (non-number) input

## Concepts Used
- Python fundamentals
- Loops
- Conditional statements
- Functions
- random module
- try-except (basic error handling)

## How to Run
1. Make sure Python is installed
2. Open terminal in project folder
3. Run:
   python GuessTheNo.py
